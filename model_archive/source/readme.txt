Source code references:
-----------------------

	GSFLOW version 2.3.0
	
		Documentation citation:
		Markstrom, S.L., Niswonger, R.G., Regan, R.S., Prudic, D.E., and Barlow, P.M., 2008, 
			GSFLOW-Coupled Ground-water and Surface-water FLOW model based on the integration 
			of the Precipitation-Runoff Modeling System (PRMS) and the Modular Ground-Water 
			Flow Model (MODFLOW-2005): U.S. Geological Survey Techniques and Methods 6-D1, 
			240 p., https://doi.org/10.3133/tm6D1
		
		Download from:
		https://www.usgs.gov/software/gsflow-coupled-groundwater-and-surface-water-flow-model
	
	
	MODSIM version 8.6
	
		Documentation citation:
		Labadie, J.W., 2010, MODSIM 8.1: River Basin Management Decision Support System: User Manual and Documentation, 
			accessed at http://modsim.engr.colostate.edu/modsim.php.

		Download from:
		http://modsim.engr.colostate.edu/modsim.php
